# javascript-web-html-to-pdf

## Description
A demo of html rendering to pdf. In this demo
we can use html fragments by tagname or id only.

## Tech stack
- jspdf

## Docker stack
- httpd:latest

## To run
`sudo ./install.sh -u`
- Available at http://localhost

## To stop
`sudo ./install.sh -d`

## To see help
`sudo ./install.sh -h`
